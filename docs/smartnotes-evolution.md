# SmartNotes 演进记录

## 2026-05-14 - UI 全面优化 v2.0（完成）

### 设计决策
- **设计风格**：简约现代（大量留白、卡片式、渐变色、微交互）
- **深色模式**：支持，用户可切换
- **字体**：Inter + 苹方 + 微软雅黑

### 完成的设计改动

#### 1. 整体布局
- Header：渐变背景（靛蓝紫→紫色）+ 毛玻璃效果装饰
- Logo：图标 + 标题 + 副标题组合
- 新建按钮：纯白渐变 CTA 风格
- 主题切换：毛玻璃圆形按钮

#### 2. 配色方案

**亮色模式**
- 主色：#6366f1（靛蓝紫）
- 强调色：#f43f5e（玫红）
- 背景：#fafafa
- 卡片：#ffffff

**暗色模式**
- 主色：#818cf8（浅靛蓝紫）
- 背景：#0f172a（深蓝黑）
- 卡片：#1e293b

#### 3. 便签卡片
- 圆角：16px
- 阴影：多层柔和阴影 + hover 时彩色阴影
- 顶部渐变条（hover 显示）
- 操作按钮 hover 显示
- 内容预览限制 4 行

#### 4. 编辑器弹窗
- 居中大卡片 + 毛玻璃遮罩
- 格式工具栏重新设计
- 新增链接插入功能（下划线按钮 + 链接按钮）
- 底部快捷键提示

#### 5. 动效
- 卡片 staggered 淡入动画
- hover 上浮 + 阴影加深
- 编辑器缩放淡入
- 主题切换平滑过渡
- 按钮点击缩放反馈

#### 6. 其他优化
- 滚动条美化
- 空状态设计
- 模态框统一风格
- 响应式适配（移动端底部弹出编辑器）
- 底部操作栏现代化设计
